## 🚀 使用方法

### 1. 打开项目
- 双击 `index.html` 在浏览器中打开
- 或者使用本地服务器（推荐）

### 2. 体验功能
1. 点击"开始探索"进入地图页
2. 点击地图上的6个文物图标
3. 查看右侧显示的非遗项目详情和真实图片
4. 切换Tab标签查看不同内容
5. 勾选互动任务复选框
6. 点击"导航模式"/"漫游模式"按钮切换

   - 浮动动画（float）
   - 脉冲动画（pulse）
   - 旋转动画（rotate）
   - 淡入动画（fadeIn）
   - 滑入动画（slideIn）

2. **transition过渡**
   - 0.3s平滑过渡

   - transform变换

   - 标记点浮动（3秒循环）
   - 罗盘旋转（4秒循环）

### JavaScript交互
